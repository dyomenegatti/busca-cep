# Buscar CEP

Projeto desenvolvido em HTML, CSS e JavaScript para pesquisa de CEP utilizando a API [ViaCEP](https://viacep.com.br/).

## Instalação

Não é necessário nenhum tipo de instalação. Basta clonar ou baixar o repositório e abrir o arquivo **index.html** em um navegador web compatível.

## Uso
1. Abra o arquivo **index.html** em um navegador web.
2. Insira um CEP no campo de pesquisa e pressione Enter ou clique no botão "Buscar CEP".
3. Os dados correspondentes ao CEP serão exibidos abaixo do formulário de pesquisa.

## Funcionalidades
- Pesquisa de CEP utilizando a API ViaCEP.
- Exibição de dados como logradouro, bairro, complemento, cidade e estado correspondentes ao CEP inserido.
